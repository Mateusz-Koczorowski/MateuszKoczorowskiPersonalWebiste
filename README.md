# Senior QA Engineer Personal Website

Production-ready starter repository for a personal technical brand website focused on senior QA engineering, test automation, AI-assisted QA workflows, quality strategy, technical case studies, and community work.

## Stack

- Next.js App Router with TypeScript
- Tailwind CSS v4
- shadcn/ui-compatible component structure
- Framer Motion
- ESLint and Prettier

## Getting Started

```bash
npm install
npm run dev
```

## Project Structure

```text
app/             Next.js routes, metadata, global styles
components/      Reusable UI and page components
content/         Typed content primitives and future content models
docs/            Product, brand, UX, architecture, AI, and backlog docs
docs/stories/    Implementation stories for incremental delivery
public/          Static assets
tests/           Test strategy and future automated checks
```

## Quality Bar

- Semantic HTML and accessible interactions by default
- Mobile-first responsive layouts
- SEO-ready metadata, robots, and sitemap foundations
- No lorem ipsum, fake metrics, confidential project names, or unverified claims
- Public case studies must be sanitized and written from reusable technical patterns

## Scripts

- `npm run dev` starts the local development server
- `npm run build` creates a production build
- `npm run lint` runs ESLint
- `npm run typecheck` runs TypeScript without emitting files
- `npm run format:check` validates formatting
