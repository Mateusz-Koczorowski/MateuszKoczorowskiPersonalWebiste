# Story 005: Hero Design Sandbox

## Goal

Create the first visual prototype of the homepage hero section using the existing dark technical premium design foundation.

## Scope

- Dedicated sandbox route at `/design-sandbox/hero`.
- Hero section only.
- Profile image path: `/images/profile/profile-photo.jpg`.
- Large name, role, positioning statement, and three CTA links.
- Desktop split layout with image on the left and content on the right.
- Mobile layout with image above content.

## Acceptance Criteria

- The prototype uses semantic `main` and `section` landmarks.
- The hero heading identifies `Mateusz Koczorowski`.
- CTA elements are native links and keyboard focusable.
- Layout is responsive from mobile to desktop widths.
- Motion uses the existing reduced-motion-aware `AnimatedSection` component.
- No full homepage sections or fake case-study content are added.

## Preview

Run `npm run dev` and open `http://localhost:3000/design-sandbox/hero`.
